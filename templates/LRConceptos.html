Conceptos Básicos de la Regresión Lineal

El análisis de regresión lineal es una técnica de análisis de datos que se utiliza para predecir el valor de una variable en función del valor de otra variable. La variable que se desea predecir se denomina variable dependiente, mientras que la variable que se utiliza como referencia se conoce como variable independiente.

Esta forma de análisis ajusta una línea recta (o un plano en multivariables) que minimiza la diferencia entre los valores reales observados y los valores predichos por el modelo. El método más usado es el de mínimos cuadrados.

📌 Linealidad

La relación entre las variables independientes y la dependiente debe ser lineal. Se verifica con gráficos de dispersión o residuos frente a los valores ajustados. Si no se cumple, el modelo pierde capacidad predictiva.

📌 Independencia

Los errores no deben estar correlacionados entre sí. Se comprueba con la prueba de Durbin-Watson. Si no se cumple, aparece autocorrelación y la inferencia es poco fiable.

📌 Homoscedasticidad

La varianza de los errores debe ser constante para todos los valores de las variables. Si no se cumple, aparece heterocedasticidad que distorsiona los errores estándar.

📌 Normalidad del error

Los residuos deben seguir una distribución normal con media cero. Se evalúa con histogramas o pruebas como Shapiro-Wilk. En muestras grandes, la falta de normalidad tiene menor impacto gracias al TCL.
